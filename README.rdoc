Nomster 

Nomster was created using the Ruby on Rails framework. Nomster is a Yelp clone, in which users can create a profile, post listings for restaurants they enjoy, as well as rate, comment on, and upload photos for existing listings. 

Features: 
  - Google Maps API is integrated into each listing 
  - Twitter Bootstrap 3 
  - Devise 
  - AWS (S3) is utilized for media storage (user photos)
